# 07 - 技术方案

## 7.1 技术栈

### 前端

| 项目 | 选择 | 理由 |
|---|---|---|
| 框架 | React | 生态最完善，Cursor 最熟 |
| 构建工具 | Vite | 快、配置简单 |
| 路由 | React Router | 标准选择 |
| 样式 | Tailwind CSS | 快速开发 |
| 状态管理 | Zustand 或 React Context | 项目规模适中，不需要 Redux |
| 表单 | React Hook Form | 表单多，需要好用的库 |
| 图片处理 | html2canvas（分享卡）+ browser-image-compression（压缩）| |
| MD5 计算 | crypto-js | 截图 MD5 比对 |

### 后端

| 项目 | 选择 | 理由 |
|---|---|---|
| 数据库 | Firebase Firestore | 实时同步、免运维、便宜 |
| 文件存储 | Firebase Storage | 图片存储 |
| 认证 | Firebase Auth | 手机号验证码 |
| 后端逻辑 | Cloud Functions（仅必要时）| 大部分逻辑在前端做 |
| 部署 | Vercel | 静态站点 + Serverless |

### PWA

| 项目 | 选择 |
|---|---|
| Service Worker | Workbox（Vite PWA Plugin）|
| 推送通知 | Firebase Cloud Messaging |

## 7.5 关键技术细节

### 7.5.1 顾客身份识别（无注册场景）

游客没有账号，但需要在浏览器里"记住"自己的订单。

**方案：localStorage + 订单 ID 列表**

```typescript
// 下单成功后
const localOrders = localStorage.getItem('myOrders') || '[]';
const orders = JSON.parse(localOrders);
orders.push({
  orderId: 'xxx',
  shopSlug: 'huijie',
  projectId: 'yyy',
  createdAt: new Date().toISOString(),
});
localStorage.setItem('myOrders', JSON.stringify(orders));

// 查看"我的订单"时
const orders = JSON.parse(localStorage.getItem('myOrders') || '[]');
const projectOrders = orders.filter(o => o.projectId === currentProjectId);
// 用 orderIds 查询 Firestore
```

**注意事项：**
- 清理浏览器缓存会丢失（提醒顾客注册可解决）
- 不同浏览器/设备数据独立
- 微信内打开可获取 openId 作为强标识

### 7.5.2 实时同步

使用 Firestore 的 `onSnapshot`：

```typescript
// 监听项目状态（顾客端）
useEffect(() => {
  const unsub = db.collection('projects').doc(projectId)
    .onSnapshot(doc => {
      setProject(doc.data());
    });
  return unsub;
}, [projectId]);

// 监听新订单（商户端）
useEffect(() => {
  const unsub = db.collection('orders')
    .where('projectId', '==', projectId)
    .where('status', '==', 'pending')
    .onSnapshot(snapshot => {
      // 处理变化
      snapshot.docChanges().forEach(change => {
        if (change.type === 'added') {
          // 播放提示音、显示通知
          playNotificationSound();
        }
      });
    });
  return unsub;
}, [projectId]);
```

### 7.5.3 MD5 比对

```typescript
import CryptoJS from 'crypto-js';

async function calculateMD5(file: File): Promise<string> {
  return new Promise((resolve) => {
    const reader = new FileReader();
    reader.onload = () => {
      const wordArray = CryptoJS.lib.WordArray.create(reader.result as ArrayBuffer);
      const md5 = CryptoJS.MD5(wordArray).toString();
      resolve(md5);
    };
    reader.readAsArrayBuffer(file);
  });
}

// 上传付款截图时
const md5 = await calculateMD5(file);

// 检查重复
const duplicates = await db.collection('orders')
  .where('shopId', '==', shopId)
  .where('paymentScreenshots', 'array-contains-any', [{ md5Hash: md5 }])
  .get();

const flag = duplicates.empty ? 'green' : 'red';
```

### 7.5.4 库存防超卖

使用 Firestore Transaction：

```typescript
async function createOrder(orderData) {
  return await db.runTransaction(async (transaction) => {
    const projectRef = db.collection('projects').doc(orderData.projectId);
    const project = await transaction.get(projectRef);
    
    // 检查每个商品的库存
    for (const item of orderData.items) {
      const product = project.data().products.find(p => p.id === item.productId);
      if (product.stock < item.quantity) {
        throw new Error(`${product.name} 库存不足`);
      }
    }
    
    // 扣减库存
    const updatedProducts = project.data().products.map(p => {
      const orderedItem = orderData.items.find(i => i.productId === p.id);
      if (orderedItem) {
        return { ...p, stock: p.stock - orderedItem.quantity };
      }
      return p;
    });
    
    transaction.update(projectRef, { products: updatedProducts });
    
    // 创建订单
    const orderRef = db.collection('orders').doc();
    transaction.set(orderRef, orderData);
    
    return orderRef.id;
  });
}
```

### 7.5.5 分享卡片生成

```typescript
import html2canvas from 'html2canvas';

async function generateShareCard(projectData) {
  // 1. 创建一个隐藏的 div，渲染卡片内容
  const cardDiv = document.createElement('div');
  cardDiv.style.position = 'absolute';
  cardDiv.style.left = '-9999px';
  cardDiv.style.width = '1080px';
  cardDiv.style.height = '1440px';
  // ... 渲染卡片样式
  document.body.appendChild(cardDiv);
  
  // 2. 用 html2canvas 截图
  const canvas = await html2canvas(cardDiv, {
    width: 1080,
    height: 1440,
    scale: 2,
  });
  
  // 3. 转 Blob 让用户下载
  canvas.toBlob((blob) => {
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `${projectData.title}-分享卡.png`;
    a.click();
  });
  
  // 4. 清理
  document.body.removeChild(cardDiv);
}
```

## 7.7 性能优化建议

1. **图片懒加载** — 商品图、菜品图等用 `loading="lazy"`
2. **路由懒加载** — 用 `React.lazy()` 拆分代码
3. **图片压缩** — 用户上传图片前自动压缩（用 browser-image-compression）
4. **Firestore 数据缓存** — 列表页面缓存查询结果
5. **缓存订单数据** — Service Worker 缓存最近的订单数据

## 7.8 移动端优化

- 所有点击区域不小于 44x44px（iOS 触摸标准）
- 表单输入触发数字键盘（手机号、价格）
- 长按图片可保存（付款码）
- 支持微信内分享 API（如果在微信内）
- 考虑安全区域（iPhone 刘海屏、底部 Home 条）
